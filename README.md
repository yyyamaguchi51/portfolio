# アプリ概要
PHPとMySQLを使用した、社員情報の登録・管理を行うアプリです。

## 制作背景
前職での入社手続きにおいて、同じ情報を何度も入力する手間を改善したいという実体験から制作しました

## 機能一覧
*   社員情報の登録・編集・削除
*   ログイン・ログアウト機能
*   郵便番号による住所自動入力（ajaxzip3）

## 使用技術
*   言語：PHP 8.x
*   データベース：MySQL
*   サーバー環境：XAMPP (Windows)
*   ライブラリ：ajaxzip3

## 工夫したところ
*   セキュリティ：パスワードをハッシュ化して保存し、安全性を確保
*   利便性：Ajaxを用いた郵便番号自動入力を実装し、入力負担を軽減
*   データ活用：登録情報をCSVなどでダウンロードできる機能を付け、2次利用を可能に
